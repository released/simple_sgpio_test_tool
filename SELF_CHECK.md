## Hardware Checks

- Use GUI `Get Info`, `Enable`, `Apply Now`, `Get Status`, and `Disable`.
- Use a logic analyzer on PA2/PA0/PA3 to verify SGPIO frame timing.
- Test against:
  - `M031BSP_Software_SGPIO`
  - `M031BSP_SPI_SGPIO`

## Known Limits

- M032 firmware currently reports PA1 level as SDATA in status but does not capture SDATA-in frame words.
- SGPIO is generated by Timer0-triggered PDMA writes to PA DOUT; final timing should still be checked with a logic analyzer on the actual M032 EVB.
